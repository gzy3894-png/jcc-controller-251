## 版本号规则（从 v1.1.0 起）

```text
v主.次.修订
例：v1.1.0
```

- **主版本**：注入/打包方式大改  
- **次版本**：功能  
- **修订**：修 bug / 日志  

旧标签（`full-1.0.x`、`2.6.x` 等）仅作历史，**不要再装**。

## 安装

1. 下载 **Latest** 的 `JCC-Controller.apk`  
2. 覆盖安装  
3. 打开 App → **一键修复** → **一键启动**  
4. 日志只看：`/data/user/0/com.tencent.jkchess/files/log.txt`  

日志里应出现当前版本标记（如 `[FULL-1.1.0]`）。
